Add instant "Call Now" and "Chat" buttons to your website, allowing visitors to seamlessly contact you with a single click.

== Description ==

With the "Call Now and Chat Button" plugin, give your website visitors an effortless way to reach out to you. Perfect for businesses, e-commerce stores, bloggers, and any website where customer communication is key. You can also utilize our "Call us" button independently, even with a different color, without relying on WhatsApp.

**Features**:

- Instant "Call Now" button for immediate phone calls.
- Direct "Chat" button for quick chats.
- Seamless integration with any WordPress theme.
- User-friendly settings for easy customization.
- Option to display WhatsApp QR code on desktop for easy access.
- Flexibility to disable the WhatsApp button, allowing the Call Us button to occupy full width.
- Customizable button colors for better branding.

== Installation ==

1. Upload the `call-now-and-chat-buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings as desired.

== Frequently Asked Questions ==

= Is this plugin mobile-responsive? =
Yes, the "Call Now" and "Chat" buttons are designed to work on both desktop and mobile devices.

= Can I customize the appearance of the buttons? =
Yes, you can customize the buttons' appearance from the plugin settings.
